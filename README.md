GTPS2FBITool: A addon for blender to import and export GT3 (.FBI) scene files.

- GT3's .FBI scene files are pre-baked meaning the animations are baked into the engine and the .FBI acts as the operator for the parameters. 
- How to install:
Edit > Preferences > Add-on > Pick the *GTPS2FBITool.Zip* and tick the checkbox > Save preferences > exit.

Requirements:
- Basic Blender Knowledge.
